Mobile App

Mobile Framework

This is a mobile framework using HTML5, CSS3, and Javascript. Using a single page HTML file, tabs are incorporated to create pages with transitions to move between them.

Geolocation API is used to display a google static image of the users current location.

The contacts page uses the Cordova Contacts API which fetches and displays one random contact from the contacts on the mobile device.
